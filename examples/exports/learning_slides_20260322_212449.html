## What Is This Topic About?

The **Transformer architecture** (used in GPT, BERT, and virtually every modern AI system) relies on a mechanism called **self-attention** — a powerful way for models to understand relationships between all parts of an input sequence.

When we train an LLM, the loss function used is called **Cross-Entropy Loss**.

At each step, the model looks at some text and tries to predict the next token. Cross-entropy measures **how wrong the model's prediction was**:

  "myocardial"  → 0.45  ← Model thinks this is most likely
  "kidney"      → 0.20
  "liver"       → 0.15
  "chronic"     → 0.10
  ... (other tokens) → 0.10

Actual next word: "myocardial" ✅

Cross-entropy loss = -log(0.45) ≈ 0.799
